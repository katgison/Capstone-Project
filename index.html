UI SCHOLARSHIP CONNECT

UI Scolarship Connect is an app that connects students with scholarship providers and other students who have successfully recieved scholarships, allowing them to share tips and advice how to succeed in the scholarship application process. The purpose of this project is to organize the system of scholarship, to make it easier for students and CSDL'S staffs.

  • This project having benefits for students especially when they process the application.
  • However, not only the students can benefit for this also the admin,staffs and teachers. this also serve as a beneficial application especially for students with scholarship.
  • Addtionally, it can create an accessible platform for both scholarship providers and students.

On other hand, this project will be effecient for students with scholarships for easy to access when they process the application. By using this app it can be immediate to process and time consuming.
